## Implementation Notes

This project (Sediment) is an original generative art system inspired by the techniques and approaches of the above artists. It is not a reproduction or copy of any specific artwork, but rather a synthesis of observed techniques:

- Rayner's multi-texture algorithms and algorithmic distress
- Kane's multi-layered optical complexity
- De Giuli's emergent systems and complex patterns

All code is original and written specifically for this project.
